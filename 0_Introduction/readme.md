书签 a 夹在了第100页
```
每次你打开a的时候。你就到了第100页
第100页就是书签a指向的位置
第100页上有1块钱
1快钱就是位置上的值／

page_100 = "ahahaha"
a = page_100
print(a)
那么print（变量 a）就是按着a找到它指向的位置 page_100
这时候page_100写着 “ahahahaha”
那么print（a）
```
以上：same as print
    “ahahaha”
